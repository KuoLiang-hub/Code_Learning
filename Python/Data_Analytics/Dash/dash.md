# Intro
python dash is 
- open source python framework for data visualization
- based on Flask, React, and Plotly

Core Technologies
- Flask python Micro-Framework
    web-browser functionality
- React.js
    User interfaces and interactivity
- Plotly
    Chart and graphics generation
